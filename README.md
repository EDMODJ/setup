# Setup

Script to install base environment for Ruby development.

## Install on Mac OS X

```
./mac.sh
```

## Installed libraries

The script will install:

* homebrew
* imagemagick
* oh-my-zsh
* mysql
* postgres
* redis
* rvm
* ruby-2.1.2
* nvm
* qt
* heroku-toolbelt
